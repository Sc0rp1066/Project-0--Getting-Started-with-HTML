Webpages, Documents, and Structure Summary:

Webpage Basics:

• The Web – (World Wide Web) is collection of HTML documents.
• HTML (Hypertext Markup Language) - The standard markup language used to create and format the content of web pages.
• Web Page Layout- Invisible boxes with content and style formatting. Class attributes define the content of the boxes.
• CSS - Cascading Style Sheets - Style sheet language used for describing the look and formatting of a document written in markup language.
• DOM- Document Object Model - A cross platform and language independent convention for representing and interacting with objects in HTML. The nodes of every document are organized in a tree structure called the DOM.
• URL – Link or Webpage address in the internet.
• HTTP – Hyper Text Transfer Protocol – How web rowsers talk to the webpage servers across the internet.
• Web applications

HTML is the heart of the Web:

HTML Basics:

• Tag – HTML tag is always contained within brackets made up the less than / greater than symbols. Most have and opening tag < > and closing < / >.
• Element – HTML element is everything within and including the set of opening and closing tags.
• Attribute – The property of and HTML element ex. <a href= “url”> content </a>.
• Text content (what you see)
• Markup (what the content looks like)
• References to other documents (ex. Images, videos…)
• Links to other pages.

HTML Document Structure:

<! Doctype HTML>
<HTML>
<Head>
<title> Title </title>
</Head>
<Body>
<b> Content </b>
</Body>
</HTML>

TAG Types:

Inline Tags: Only contains text. HTML renders white space to a single space.

• Anchor - <a href= “url”> content </a> - Create hyperlinks
• Break - New line (void tag has no content. Doesn’t require closing tag)
• Bold - <b> Content </b> - Bolds content
• Emphasize - <em> content </em> - Italicize Content
• Span - <span> content </span> - contains content with attribute
• Strong - <strong> content </strong> - contains content. Bolds with attribute.

Block Tag: Creates “invisible boxes” around content which can have height and width elements.

• Paragraph - <p> content </p> - Paragraph with line break
• Div - <div> content </div>
• Form - <form> content </form>
